## 📊 180-Question Framework (Golden Rule)

```
EASY (Q1-Q60)         MEDIUM (Q61-Q120)      HARD (Q121-Q180)
├─ 60 Questions       ├─ 60 Questions        ├─ 60 Questions
├─ 20% Weight         ├─ 30% Weight          ├─ 50% Weight
├─ 1-2 min each       ├─ 2-3 min each        ├─ 3-5 min each
└─ ~90 min total      └─ ~120 min total      └─ ~180 min total

TOTAL: 180 Questions, ~270 minutes
PASSING: 70% (typically)
```

## 🚨 Common Mistakes to Avoid

```
❌ Wrong: Using only 100 questions
❌ Wrong: All multiple choice (no variety)
❌ Wrong: Questions too similar to each other
❌ Wrong: No explanation in answer key
❌ Wrong: Unclear or ambiguous wording

✅ Right: Always 180 questions (60-60-60)
✅ Right: Mix question types
✅ Right: Variety in scenarios
✅ Right: Detailed explanations
✅ Right: Clear, professional wording
```

## 📞 Quick Help

**Generator not working?**
- Ensure Assessment Name is filled
- Check content percentages add up
- Try refreshing page

**Prompt too long?**
- It's supposed to be (3000+ words)
- AI systems can handle it
- Paste entire thing

**Generated questions too easy/hard?**
- Regenerate with updated objectives
- Ask AI to "increase difficulty"
- Have SME adjust manually

**Duplicates found?**
- Normal with large datasets
- Ask AI to check for duplicates
- Remove manually before use
